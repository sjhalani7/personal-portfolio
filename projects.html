February 2023

  • Collaborated with a team of 5 to optimize the performance & functionality of the web application in a 24-hour period

Project Pineapple- Mobile Check-in App

Hackathon Project - 2nd Place Winner, Hack4Humanity 2022

February 2022

  • Designed application backend using Python, Flask, and SQLite that uses scheduled check-ins & user verification to allow a safe night out to reduce the likelihood of sexual assaults
  • Implemented an SQLite database using Python programming to store & maintain data of users effectively & systematically
  • Developed an API using Flask to fetch data from database & link to the front-end software for an efficient full-stack software

I played a key role in the development of Project Pineapple, a safety-oriented initiative designed to enhance security during night outings. My primary responsibility was designing the application's backend, employing Python, Flask, and SQLite. This backend was integral to the project's core functionality: it managed scheduled check-ins and user verifications, aiming to reduce the likelihood of sexual assaults.

I implemented an SQLite database, leveraging my Python programming skills to ensure systematic and effective data storage and maintenance of user information. Additionally, I developed a Flask API to seamlessly fetch data from this database and connect it with the front-end software, contributing to a robust full-stack application.

Project Pineapple was more than just a technical endeavor; it was a profound learning experience for my first ever hackathon. It honed my skills in backend development and database management, and significantly enhanced my ability to collaborate effectively within a team, all while addressing a critical social issue.
