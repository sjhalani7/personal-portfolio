MY INTERNSHIP EXPERIENCE


I am fortunate to have landed numerous internships during my professional career thus far.

This page will walk through my experiences, highlighting some amazing things I was able to learn during my time at some incredible companies.

Backend SaaS Software Engineering Intern

INRIX Inc.

June 2023 - September 2023

  • Implemented automated multithreaded data ingestion pipeline for an internal tool using Java, Python, and AWS technologies
  • Mitigated error-prone manual process, improving efficiency by 20%, deploying code to production, & testing in CI/CD pipeline
  • Led 11-member team to final round of company-wide hackathon, through global cross-team collaboration & project management

I spent the summer at INRIX working on SafetyView, one of INRIX's IQ SaaS applications, developing meaningful, impactful projects, and couldn't be happier with how the summer went.

I mainly worked on implementing a multithreaded, automated data update pipeline for an internal tool via APIs and utilizing technologies like AWS Lambda and AWS Step Functions. The project not only added value to the team, by speeding up and improving quality, but also enabled me to gain a massive understanding of serverless computing, large-scale API development, and most importantly, software engineering in the industry.

The highlight of my internship was INRIX's Innovation Week: everyone takes a week off their regular tasks and participates in a company wide hackathon to brainstorm new ideas and bring fresh perspectives to the company. I was fortunate to have been a part of this incredible experience, and went so far as to create my own team, leading 12 full-time employees from all around the world to the final round of the hackathon.

Data Science Intern

Ernst & Young LLC

June 2022 - August 2022

  • Researched, designed, and implemented voice-enabled dashboard for data insights using natural language processing & Python
  • Utilized spaCy, pandas, sent2vec, & matplotlib libraries for natural language translation to SQL for data visualization
  • Developed strong foundation for analytics tool that optimizes data visualization for Chief Experience Officer insights

During my summer at Ernst and Young, I led a project from start to finish for the first time. I was granted the freedom to design and execute the project, blending my knowledge of pandas and matplotlib with new technologies like spaCy and sent2vec for natural language processing.

A significant part of this experience was the combination of planning and hands-on implementation, which gave me a peek into the field of project management. I particularly enjoyed researching and developing a method to convert natural language into SQL queries and visualizing the results. This internship was a practical learning journey that enhanced my technical and problem-solving skills.

Summer Intern

My summer with NTT Data Services in June 2019 was my first internship experience. As a junior in high school, with programming being a relatively new concept, this internship was a tipping point for my journey with computer science, solidifying the passion that was ignited by a Java course a couple years prior.
